## Quick Start

- Install Dependencies

  ```bash
  pip install -r requirements.txt
  ```

- Download Data

  ```bash
  cd SentEval/data/downstream/
  bash download_dataset.sh
  cd -
  cd ./data
  bash download_nli.sh
  cd -
  ```

- Python version 3.9.18
